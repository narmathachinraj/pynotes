```python
text = input("Enter text: ")
digits = letters = special = 0

for ch in text:
    if ch.isdigit():
        digits += 1
    elif ch.isalpha():
        letters += 1
    else:
        special += 1

print("Digits:", digits)
print("Letters:", letters)
print("Special:", special)
```

    Enter text:  sanju
